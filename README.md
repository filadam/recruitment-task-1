# Zadanie rekrutacyjne

## Wymagania

Node.js

[Node](http://nodejs.org/) zawiera także npm wymagany do odpalenia projektu

    $ node --version
    v0.10.24

    $ npm --version
    1.3.21

## Instalacja

    $ git clone https://github.com/filadam/zadanierekrutacyjne-1.git
    $ cd PROJEKT
    $ npm install
    
## Start

    $ npm start
